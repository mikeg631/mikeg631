Getting Started:
To run you must first generate java code from the gammar using a command like  "antlr4 BC.g4".
Next compile the with "javac *.java". Make sure that the classpath has the antlr4 library. COMPILE ALL JAVA FILES IN THIS DIRECTORY
Run using exprList as the top rule: grun BC exprList  testFile.bc

Make sure to run PrecTest.bc and arrayTest.bc
